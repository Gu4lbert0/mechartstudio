Project case study

Fruit Counting System

Computer vision based yield estimation platform utilizing depth cameras, GPS, and mapping technology.

Back to Projects

Project Summary

The Fruit Counting System is an agricultural sensing and yield estimation platform built around computer vision, depth sensing, GPS, and mapping workflows. The system is intended to help estimate crop yield and organize field data into useful information for growers and engineering teams.

The platform combines physical mounting, field-ready sensor placement, data collection planning, and software-facing integration needs so the sensing system can be tested in realistic orchard conditions.

My Role

  • Supported mechanical design decisions for sensor mounting and field deployment.
  • Contributed to integration planning for cameras, GPS, and supporting electronics.
  • Helped bridge hardware constraints with mapping and data collection requirements.
  • Focused on practical packaging, test readiness, and field usability.
