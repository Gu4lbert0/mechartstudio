Object Counting and Mapping

A handheld robotic vision system designed to detect, count, and generate three-dimensional maps of objects by combining computer vision, depth sensing, visual-inertial odometry, and embedded computing into a single portable platform.

Back to Projects

Project Summary

The Object Counting and Mapping project focused on developing a handheld robotic vision system capable of detecting, counting, and generating three-dimensional locations of objects in real time. The platform integrates stereo vision, depth sensing, inertial measurement units (IMUs), and onboard computing to estimate the position of detected objects relative to the surrounding environment. Throughout development, multiple localization methods were researched and evaluated, beginning with an RTK GPS-based solution before transitioning to Kimera-VIO to improve camera tracking and mapping performance through visual-inertial odometry.

My Role

  • Designed the handheld enclosure and mechanical components using CAD software.
  • Selected and integrated cameras, sensors, embedded computers, and communication hardware.
  • Developed software for object detection, tracking, and three-dimensional localization.
  • Implemented sensor fusion using stereo vision and IMU data to improve mapping accuracy.
  • Researched, implemented, and evaluated both RTK GPS and Kimera-VIO to determine the most effective localization solution for the handheld platform.
  • Designed and assembled the electrical system, including wiring and power distribution.
  • Collected and analyzed field data to improve localization accuracy and object detection reliability.
  • Continuously refined both the hardware and software through iterative prototyping and testing.
