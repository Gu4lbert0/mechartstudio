# M-Robotics

Static website and project portfolio for M-Robotics, deployed at `mechartstudio.com`.

## Structure

- `index.html` - Home page
- `about.html` - Company overview
- `projects.html` - Project portfolio
- `services.html` - Engineering services
- `contact.html` - Static contact page
- `css/style.css` - Shared responsive styles
- `js/main.js` - Mobile navigation and form UI behavior
- `images/` - Local site imagery
